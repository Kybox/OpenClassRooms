Créer un dossier :
> mkdir mon_dossier

Se déplacer dans un dossier :
> cd mon_dossier

Activer un dossier comme repository :
(se placer préalablement dans le dossier)
> git init

Ajouter tout les fichiers à l'index Git :
> git add .

Enregistrer un commit :
> git commit -m "Message d'information"

Lire l'historique :
> git log

Mettre à jour un repository en local à partir de Git-Hub :
> git pull origin master

Mettre à jour un repository sur Git-Hub à partir d'un ordinateur :
> git push origin master
